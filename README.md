# 🛒 Retail Sales Data Analysis & Business Insights

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on a retail sales dataset to uncover business insights such as sales trends, profit distribution, and category performance.

---

## 🎯 Objectives
- Analyze overall sales and profit
- Identify top-performing categories
- Detect loss-making segments
- Understand regional performance
- Study monthly sales trends
- Generate business insights

---

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

---

## 📊 Key Insights
- Technology category generates highest profit
- Some sub-categories result in losses
- Sales vary significantly across states
- Monthly trends show seasonal variation
- Profit is not always proportional to sales
